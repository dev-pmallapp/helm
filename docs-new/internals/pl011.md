# PL011

ARM PrimeCell UART (PL011) device model.

## Register Map

| Offset | Name | R/W | Description |
|--------|------|-----|-------------|
| 0x000 | UARTDR | RW | Data register |
| 0x004 | UARTRSR | RW | Receive status / error clear |
| 0x018 | UARTFR | R | Flag register |
| 0x028 | UARTIBRD | RW | Integer baud rate divisor |
| 0x02C | UARTFBRD | RW | Fractional baud rate divisor |
| 0x030 | UARTLCR_H | RW | Line control |
| 0x034 | UARTCR | RW | Control |
| 0x038 | UARTIFLS | RW | Interrupt FIFO level select |
| 0x03C | UARTIMSC | RW | Interrupt mask set/clear |
| 0x040 | UARTRIS | R | Raw interrupt status |
| 0x044 | UARTMIS | R | Masked interrupt status |
| 0x048 | UARTICR | W | Interrupt clear |
| 0xFE0–0xFFC | PeriphID/CellID | R | Identification |

## CharBackend

The PL011 connects to a `CharBackend` for actual I/O:

- `StdioCharBackend` — reads/writes to host stdin/stdout.
- `NullCharBackend` — discards output, returns no input.
- `BufferCharBackend` — in-memory buffer for testing.

## Flag Register Bits

| Bit | Name | Description |
|-----|------|-------------|
| 3 | BUSY | Transmitting |
| 4 | RXFE | Receive FIFO empty |
| 5 | TXFF | Transmit FIFO full |
| 7 | TXFE | Transmit FIFO empty |

## FIFO

The PL011 model uses a `VecDeque<u8>` for the receive FIFO.
Transmit writes go directly to the `CharBackend`.
