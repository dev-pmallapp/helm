# BCM2837

Raspberry Pi 3 peripheral device models.

## Peripherals

| Module | Device | Register Base |
|--------|--------|---------------|
| `bcm_sys_timer` | System Timer | 0x3F00_3000 |
| `bcm_mailbox` | Mailbox | 0x3F00_B880 |
| `bcm_gpio` | GPIO | 0x3F20_0000 |
| `bcm_mini_uart` | Mini UART (UART1) | 0x3F21_5040 |

## System Timer

64-bit free-running counter with 4 compare channels (C0–C3).
C0/C2 are used by the GPU; C1/C3 by the ARM.

Registers: CS (control/status), CLO/CHI (counter), C0–C3 (compare).

## Mailbox

ARM ↔ VideoCore communication interface. Used for framebuffer setup,
clock configuration, and power management.

Implements property tag processing for common tags:
- `GET_BOARD_REV`, `GET_ARM_MEMORY`, `GET_VC_MEMORY`
- `GET_CLOCK_RATE`, `SET_CLOCK_RATE`
- `ALLOCATE_BUFFER`, `SET_PHYS_WH`, `SET_VIRT_WH`, `SET_DEPTH`

## GPIO

54 GPIO pins with function select (input, output, alt0–alt5),
pull-up/down control, and event detection.

Registers: GPFSEL0–5 (function select), GPSET/GPCLR (output),
GPLEV (level), GPEDS (event detect), GPPUD/GPPUDCLK (pull).

## Mini UART

16550-like UART with simplified FIFO. Registers: IO, IER, IIR, LCR,
MCR, LSR, MSR, SCRATCH, CNTL, STAT, BAUD. Connects to a
`CharBackend`.
