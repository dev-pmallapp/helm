# Glossary

| Term | Definition |
|------|------------|
| **ITE** | Interval-Timing Emulation — timing model with per-class latencies |
| **ASID** | Address Space Identifier — tags TLB entries per process |
| **CAE** | Cycle-Accurate Emulation — full pipeline timing model |
| **DAIF** | Debug, SError, IRQ, FIQ mask bits in PSTATE |
| **DTB** | Device Tree Blob — binary description of hardware topology |
| **EL** | Exception Level (0–3) in AArch64 |
| **ERET** | Exception Return instruction |
| **ESR** | Exception Syndrome Register |
| **FAR** | Fault Address Register |
| **FDT** | Flattened Device Tree (source format for DTB) |
| **FE** | Functional Emulation — IPC=1 timing model |
| **FS** | Full System mode — boots real kernels |
| **GIC** | Generic Interrupt Controller (ARM) |
| **HCR** | Hypervisor Configuration Register |
| **HOM** | HELM Object Model (inspired by QEMU QOM) |
| **ICC** | Interrupt Controller CPU interface (GICv3 sysregs) |
| **IPC** | Instructions Per Cycle |
| **IQ** | Issue Queue — scheduler for ready-to-issue uops |
| **IRQ** | Interrupt Request |
| **ISS** | Instruction-Specific Syndrome (field in ESR) |
| **ITS** | Interrupt Translation Service (GICv3) |
| **JIT** | Just-In-Time compilation (Cranelift backend) |
| **LPI** | Locality-specific Peripheral Interrupt (GICv3) |
| **LSQ** | Load/Store Queue |
| **MAIR** | Memory Attribute Indirection Register |
| **MMIO** | Memory-Mapped I/O |
| **MMU** | Memory Management Unit |
| **MOESI** | Modified/Owned/Exclusive/Shared/Invalid coherence protocol |
| **MPKI** | Mispredictions Per Kilo-Instructions |
| **NZCV** | Negative, Zero, Carry, Overflow condition flags |
| **PPI** | Private Peripheral Interrupt |
| **RAT** | Register Alias Table (rename unit) |
| **ROB** | Reorder Buffer — tracks in-flight instructions |
| **SCR** | Secure Configuration Register (EL3) |
| **SCTLR** | System Control Register |
| **SE** | Syscall Emulation mode — user-space only |
| **SPI** | Shared Peripheral Interrupt |
| **SPSR** | Saved Program Status Register |
| **SVC** | Supervisor Call instruction (syscall) |
| **TCG** | Tiny Code Generator (QEMU-style IR) |
| **TCR** | Translation Control Register |
| **TLB** | Translation Lookaside Buffer |
| **TLBI** | TLB Invalidate instruction |
| **TTBR** | Translation Table Base Register |
| **VBAR** | Vector Base Address Register |
| **VirtIO** | Virtualised I/O specification |
| **VMID** | Virtual Machine Identifier |
| **VMSA** | Virtual Memory System Architecture |
